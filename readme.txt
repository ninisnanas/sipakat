==========
PERSIAPAN
==========
1. download yii http://yii.googlecode.com/files/yii-1.1.13.e9e4a0.zip
2. ekstrak di htdocs
3. copy folder web (v2) di dropbox ke htdocs
4. jalankan via localhost/(3)

======================
KONFIGURASI (optional)
======================
kalo folder yang di ekstrak (2) namanya "yii-1.1.13.e9e4a0", ga perlu konfigurasi
5. buka file index.php di (3)
6. edit baris ke 4, ganti nama foldernya sesuai di (2)
